Privacy Policy for `IWannaRWX`

Last updated: February 23, 2026

1. Data Collection and Storage
   All your personal data is stored locally on your device only using:
   - Swift Data for app data
   - Apple Keychain for secure API key storage

We do not collect or store any data on our servers.

2. Third-Party Services

We use two external services to provide content:

A) The Movie Database (TMDB) API - for movie information
- Your API key is stored securely in Apple Keychain
- When you search for movies, requests are sent to TMDB servers
- TMDB may collect: IP address, search queries, movie requests
- TMDB Privacy Policy: https://www.themoviedb.org/privacy-policy

B) Open Library API - for book metadata
- Public API, no authentication required
- When you search for books, requests are sent to OpenLibrary servers
- OpenLibrary may collect: IP address, search queries, book requests
- OpenLibrary is a project of Internet Archive

We do not control these services' data collection practices.

3. What Data is Transmitted

When you search for movies (TMDB):
- Movie search queries
- Movie IDs for detailed information
- Your IP address (automatically by network connection)

When you search for books (OpenLibrary):
- Book search queries (title, author, ISBN)
- Book identifiers for detailed information
- Your IP address (automatically by network connection)

We do NOT send to these services:
- Your name, email, or personal information
- Device identifiers or advertising IDs
- Location data beyond IP address
- Any data stored locally in the app
- Your reading or viewing history

4. API Key Security
   Your TMDB API key is stored in Apple Keychain, which uses:
- Hardware-level encryption
- Protection by your device passcode/Face ID/Touch ID
- Automatic secure storage management by iOS

Note: OpenLibrary API does not require authentication.

5. Your Data Control
   Local data: Delete the app to remove all local data and API keys
   TMDB data: Refer to TMDB's privacy policy for their data retention
   OpenLibrary data: Refer to Internet Archive's privacy policy

6. No Analytics or Tracking
   We do not use analytics, advertising, or user tracking services.
   We cannot see what you search for, which movies you view, or which books you read.

7. Children's Privacy
   This app does not knowingly collect personal information from anyone,
   including children under 13.

8. Changes to This Policy
   We may update this policy. Changes will be posted in this document.

9. Contact Us
   Questions? Email: apple@javanix.com
